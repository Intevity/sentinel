**Sentinel** is an open-source companion for [Claude Code](https://claude.com/claude-code):
a tray app and a bundled local daemon that sit between Claude Code and Anthropic's API.
From one control panel you can scan and sandbox every request, enforce permission rules,
route across the Claude accounts you own, cut wasted tokens, and catch overage spend before
it runs away — all without your data ever leaving your machine.

<CardGrid>
  <Card title="Runs locally" icon="laptop">
    The proxy listens only on `127.0.0.1`. No telemetry, no cloud — credentials live in your OS
    keychain.
  </Card>
  <Card title="Transparent to Claude Code" icon="puzzle">
    One activation writes `ANTHROPIC_BASE_URL` into Claude Code's settings. Everything else is
    pass-through.
  </Card>
  <Card title="Cross-platform" icon="seti:html">
    macOS, Windows, and Linux. A single Tauri tray app with an embedded daemon sidecar.
  </Card>
  <Card title="Open source" icon="github">
    MIT licensed. Build it from source or grab a signed, notarized release.
  </Card>
</CardGrid>

## What's in these docs

- **Getting Started** — install, first launch, and a five-minute quick start.
- **Features** — one page per capability: accounts & routing, security scanning, permission
  rules, sandbox isolation, usage, metrics, token optimization, and alerts.
- **Guides** — task-oriented how-tos (connect Claude Code, set up round-robin, sync permissions,
